Presentación

El curso trata el diseño de códigos de interfaz para distintos medios y soportes digitales, es decir, la capa Front-end dentro de un proyecto digital. Los alumnos aprenderán a desarrollar proyectos digitales enfocándose en cómo estos se componen y construyen a través de un código limpio, claro y ordenado, utilizando lenguajes tales como HTML, CSS, SASS, Javascript entre otros.

Sobre el curso

El curso del año 2021 se desarrollo de manera online. Los profesores a cargo son Basilio Cáceres y Felipe Larraín, quienes entregan sus conocimientos presentando desde sus pantallas los slides de la presentación que se prepara para cada clase, de igual forma se muestra la práctica del conocimiento desde el programa VisualCode y Github Desktop.
